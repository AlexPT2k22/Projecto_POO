\doxysubsection{Detailed Description}
Classe que representa uma biblioteca e gerencia as suas operações. 

Esta classe implementa as funcionalidades principais de uma biblioteca, incluindo\+:
\begin{DoxyItemize}
\item Gerenciamento de livros, leitores e empréstimos
\item Persistência de dados em arquivos
\item Sistema de notificações e multas
\item Geração de relatórios
\item Reservas de livros
\item Edição de informações cadastrais 
\end{DoxyItemize}
